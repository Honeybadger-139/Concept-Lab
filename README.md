# Concept Lab

**Interactive learning from transcripts and courses.** Theory, examples, animations, and tools — built from interview and course transcripts (RAG, ML, and more).

A standalone project you can link separately in your GitHub profile or portfolio to showcase turning notes and transcripts into an interactive curriculum.

## What it is

- **Sections:** RAG Systems, Machine Learning, Advanced Learning (expandable).
- **Nodes:** One node per transcript (e.g. RAG 01–14). Each node has: Core Theory, Example, Animation/visualization, Interactive tool.
- **Stack:** Next.js 16, React 19. Runs on port 3001 by default.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3001](http://localhost:3001).

## Deploy

Deploy as its own app (e.g. Vercel: new project, root directory = this repo). No connection to the main portfolio codebase — link to this repo or the live URL from your portfolio’s **Projects** or **GitHub** section.

## Link from your portfolio

Add Concept Lab as a **project** or **notable work** in your main portfolio (e.g. a card with title “Concept Lab”, description “Interactive learning from transcripts: RAG, ML. Theory, demos, and tools.”, and link to this repo or the deployed URL). The Concept Lab code does not live inside the portfolio repo.
